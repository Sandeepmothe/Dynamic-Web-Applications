# 🎯 Guess the Number Game

A simple number guessing game built with **HTML**, **CSS**, and **JavaScript**.

## 🕹️ How to Play

- Enter a number between **1 and 100**
- Click the **Check** button
- You’ll get feedback:
  - 🔼 Too high
  - 🔽 Too low
  - ✅ Correct!
- Click **Play Again** to start a new game

## 💡 Technologies Used

- HTML
- CSS
- JavaScript
